**EJECUCIÓN DE CÓDIGO:**

1. Instalar lex y yacc desde subsistema linux:

```bash
sudo apt-get install flex bison
```

2. Ejecuta el makefile: `make`

```bash
make
```

3. Ejecuta el archivo de salida: `./tarea`

```bash
./tarea
```

4. Ejectuta chmod +x tarea para dar permisos al archivo

```bash
chmod +x tarea
```
